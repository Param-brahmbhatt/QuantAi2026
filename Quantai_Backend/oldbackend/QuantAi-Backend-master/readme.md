## Environment Variables

To run this project, you will need to add the following environment variables to your `/opt/quantai/code/settings/.env`

`DJANGO_SECRET_KEY` 
`PG_DATABASE_NAME`
`PG_DATABASE_USER` 
`PG_DATABASE_PASSWORD` 
`PG_DATABASE_HOST` 
`PG_DATABASE_PORT` 
`SOCIAL_AUTH_APPLE_KEY_ID` 
`SOCIAL_AUTH_APPLE_TEAM_ID` 
`SOCIAL_AUTH_APPLE_PRIVATE_KEY` 
`SOCIAL_AUTH_APPLE_CLIENT_ID` 
`SOCIAL_AUTH_GOOGLE_CLIENT_ID` 
`SOCIAL_AUTH_GOOGLE_SECRET` 
`SOCIAL_AUTH_FACEBOOK_APP_ID` 
`SOCIAL_AUTH_FACEBOOK_SECRET` 
`SOCIAL_AUTH_TWITTER_API_KEY` 
`SOCIAL_AUTH_TWITTER_SECRET` 
`ONE_SIGNAL_APP_ID` 
`EMAIL_HOST` 
`EMAIL_USE_TLS` 
`EMAIL_PORT` 
`EMAIL_HOST_USER` 
`EMAIL_HOST_PASSWORD` 
`EMAIL_LINK_HOST`
